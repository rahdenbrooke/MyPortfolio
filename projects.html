Pizza Project

This was my final project for C++ Programming I. The program was designed to take a customers pizza order, asking specific questions like amounts, crust types, toppings, etc. At the end, the program then displays the full order with their amounts listed. The customer is only allowed up to 5 toppings per pizza before the program finishes the order (though they can choose to end at anytime).
